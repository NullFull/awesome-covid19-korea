# awesome-nCoV-Korea

## 본 레포지토리의 목적

* 2020년 코로나바이러스 대유행과 관련한 정보 취합 및 시각화를 시도한 국내외 사이트를 취합합니다.
* 흩어져 있는 코로나바이러스 관련 사이트를 한 눈에 볼 수 있도록 정리하는 목적을 갖고 있습니다.
* 추후 공공 데이터를 일반 대중에게 데이터를 제공하는 웹 사이트를 제작하려는 개발자 및 디자이너를 위하여, 데이터를 시각화한 선례를 아카이빙하려는 목적이 있습니다.
